🎆 Introdução e Visão Geral

🎯 O que é esta loja?

Esta é uma loja virtual completa que funciona integrada com WhatsApp Business. Ela permite:

  • 🛍️ Catálogo de produtos com imagens e preços
  • 💰 Links de afiliados para Shopee, Mercado Livre e Amazon
  • 📱 Integração WhatsApp para vendas diretas
  • 🎵 Sugestões TikTok para marketing de conteúdo
  • 📱 Design responsivo (funciona em celular e desktop)

🚀 Criar Conta GitHub

🎯 Por que GitHub?

  • 🆓 Gratuito: Hospedagem 100% gratuita
  • 🔒 Confiável: Usado por milhões de desenvolvedores
  • ⚡ Rápido: Servidores otimizados globalmente
  • 🔄 Versionamento: Histórico de todas as alterações
  • 🌍 Domínio: Seu próprio subdomínio .github.io

1. Acesse github.com

Abra github.com no seu navegador

2. Clique em "Sign up"

Botão verde no canto superior direito

3. Preencha os dados

  • Username: Escolha um nome único (será parte da URL)
  • Email: Seu email principal
  • Password: Senha forte (mínimo 8 caracteres)

4. Verifique o email

GitHub enviará um email de confirmação

5. Escolha o plano gratuito

"Free" é suficiente para sua loja

⚠️ Dica importante:
Escolha um username profissional, pois ele aparecerá na URL da sua loja:
https://SEU_USERNAME.github.io/loja-whatsapp-ecommerce

🚀 Como Subir no GitHub Pages

Passo 1: Criar Repositório no GitHub

  1. Acesse github.com
  2. Clique em "New repository"
  3. Nome: loja-whatsapp-ecommerce
  4. Marque como Public

Antes de subir, você precisa editar alguns arquivos para personalizar com seu usuário:

📝 2. Arquivo: client/package.json

// Encontre esta linha: "homepage": "https://SEU_USUARIO.github.io/loja-whatsapp-ecommerce" // Substitua por: "homepage": "https://seunome.github.io/loja-whatsapp-ecommerce"

Passo 3: Abrir Terminal/Prompt

💻 Windows:

  1. Pressione Win + R
  2. Digite cmd e pressione Enter
  3. Ou clique com botão direito na pasta e "Abrir no Terminal"

🍎 Mac:

  1. Pressione Cmd + Espaço
  2. Digite "Terminal" e pressione Enter
  3. Ou use Finder > Aplicativos > Utilitários > Terminal

🐧 Linux:

  1. Pressione Ctrl + Alt + T
  2. Ou clique com botão direito na pasta e "Abrir Terminal"

Passo 4: Navegar até a pasta

# Windows: cd C:\Users\Professor\Documents\Loja # Mac/Linux: cd /caminho/para/sua/pasta/Loja
📋 Dica: Se você não souber o caminho exato:
• Arraste a pasta para o terminal
• O caminho aparecerá automaticamente

Passo 5: Comandos Git (copie e cole um por vez)

1️⃣ Inicializar repositório:

git init

Este comando cria um repositório Git na pasta

2️⃣ Adicionar todos os arquivos:

git add .

Adiciona todos os arquivos para serem enviados

3️⃣ Fazer commit inicial:

git commit -m "Loja ecommerce inicial"

Cria um "snapshot" dos arquivos com uma mensagem

4️⃣ Definir branch principal:

git branch -M main

Define "main" como branch principal (padrão atual)

5️⃣ Conectar com GitHub:

git remote add origin https://github.com/SEU_USUARIO/loja-whatsapp-ecommerce.git

⚠️ Lembre-se: Substitua SEU_USUARIO pelo seu username real!

6️⃣ Enviar arquivos:

git push -u origin main

Envia todos os arquivos para o GitHub

✅ Se tudo deu certo:
Você verá uma mensagem de sucesso e os arquivos aparecerão no seu repositório GitHub

Passo 4: Instalar gh-pages

cd client npm install gh-pages --save-dev

Passo 5: Deploy

npm run deploy

Passo 6: Configurar GitHub Pages

  1. No GitHub, vá em Settings > Pages
  2. Source: Deploy from a branch
  3. Branch: gh-pages
  4. Folder: / (root)
  5. Clique em Save
✅ Pronto! Sua loja estará disponível em:
https://SEU_USUARIO.github.io/loja-whatsapp-ecommerce

💰 Sistema de Afiliados

🔗 Como Configurar Links de Afiliados

🟡 Shopee

Como se cadastrar:

  1. Acesse shopee.com.br/affiliate
  2. Cadastre-se como afiliado
  3. Copie seu ID de afiliado

Formato do link:

https://shopee.com.br/produto?affiliate_id=SEU_ID

🔵 Mercado Livre

Como se cadastrar:

  1. Acesse mercadolivre.com.br/afiliados
  2. Cadastre-se no programa
  3. Gere links personalizados

Formato do link:

https://produto.mercadolivre.com.br/produto?affiliate=SEU_ID

🟠 Amazon

Como se cadastrar:

  1. Acesse associados.amazon.com.br
  2. Cadastre-se no programa
  3. Use sua tag de associado

Formato do link:

https://amazon.com.br/produto?tag=SEU_ID

📊 Acompanhar Resultados

  • Shopee: Painel do afiliado
  • Mercado Livre: Central do vendedor

🌍 Domínio Personalizado

🎯 Por que usar domínio próprio?

  • 🏢 Profissionalismo: minhaloja.com.br em vez de username.github.io
  • 🔍 SEO: Melhor posicionamento no Google
  • 📱 Marketing: Mais fácil de lembrar e compartilhar
  • 🎨 Branding: Fortalece sua marca

💰 Onde comprar domínio:

Provedor Preço/Ano Vantagens
Registro.br R$ 40 Oficial .com.br, confiável
GoDaddy R$ 60 Interface fácil, suporte
Namecheap R$ 80 Privacidade grátis

1. Comprar o domínio

Escolha um nome relacionado ao seu negócio

Exemplos: techstore.com.br, ofertassp.com.br, minhaloja.com.br

2. Configurar DNS

No painel do seu provedor, adicione estes registros:

Tipo: CNAME Nome: www Valor: SEU_USUARIO.github.io Tipo: A Nome: @ Valor: 185.199.108.153 Valor: 185.199.109.153 Valor: 185.199.110.153 Valor: 185.199.111.153

3. Configurar no GitHub

  1. Vá em Settings > Pages no seu repositório
  2. Em "Custom domain", digite seu domínio
  3. Marque "Enforce HTTPS"
  4. Aguarde 24h para propagação

🔒 Certificado SSL

🛡️ Por que SSL é importante?

  • 🔒 Segurança: Dados criptografados
  • 🔍 SEO: Google prioriza sites HTTPS
  • ✅ Confiança: Cadeado verde no navegador
  • 📱 Compatibilidade: Funciona em todos os dispositivos
✨ Boa notícia!
O GitHub Pages oferece SSL gratuito automaticamente!

🔧 Como ativar:

1. Acesse Settings > Pages

No seu repositório GitHub

2. Marque "Enforce HTTPS"

Força o uso de HTTPS em todas as páginas

3. Aguarde alguns minutos

O certificado será gerado automaticamente

⚠️ Importante:
Se você usar domínio personalizado, aguarde 24h após configurar o DNS antes de ativar o SSL

  • 📱 Dispositivos: Mobile vs Desktop

1. Criar conta Google Analytics

  1. Acesse analytics.google.com
  2. Clique em "Começar a medir"
  3. Configure uma propriedade para sua loja
  4. Copie o código de acompanhamento

2. Instalar na loja

Adicione este código no arquivo client/public/index.html:

<!-- Google Analytics --> <script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script> <script> window.dataLayer = window.dataLayer || []; function gtag(){dataLayer.push(arguments);} gtag('js', new Date()); gtag('config', 'GA_MEASUREMENT_ID'); </script>

Substitua GA_MEASUREMENT_ID pelo seu ID real

🔍 SEO e Otimização

🎯 O que é SEO?

SEO (Search Engine Optimization) é o conjunto de técnicas para melhorar o posicionamento da sua loja no Google e outros buscadores.

🔧 Otimizações já incluídas:

  • ✅ Responsivo: Funciona em mobile
  • ✅ Velocidade: Carregamento rápido
  • ✅ HTTPS: Certificado SSL
  • ✅ Meta tags: Título e descrição

📝 Como melhorar ainda mais:

1. Títulos otimizados

Edite o arquivo client/public/index.html:

<title>Sua Loja - Produtos com Desconto | WhatsApp</title> <meta name="description" content="Encontre os melhores produtos com desconto. iPhone, Samsung, notebooks e muito mais. Compre pelo WhatsApp com segurança.">

2. Palavras-chave nos produtos

Use palavras que as pessoas pesquisam:

  • "iPhone 15 barato"
