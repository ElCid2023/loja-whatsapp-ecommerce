# 🚀 Guia Completo - Hub de Afiliados - NovidadeSP e Achados

## 📋 Visão Geral

O **Hub de Afiliados** é uma ferramenta poderosa para maximizar suas vendas através de marketing de afiliados. Com links para Shopee, Mercado Livre e Amazon, além de sugestões de conteúdo para TikTok, você pode criar uma estratégia completa de monetização.

## 🎯 Como Acessar

1. Acesse a loja: `http://localhost:3000`
2. Clique em **"💰 Afiliados"** no menu de navegação
3. Ou acesse diretamente: `http://localhost:3000/affiliate`

## 📦 Aba "Produtos com Links"

### O que você encontra:
- **15 produtos** com links de afiliados prontos
- **Links para 3 plataformas**: Shopee, Mercado Livre e Amazon
- **Palavras-chave TikTok** para cada produto
- **Preços e categorias** organizados

### Como usar:

#### 1. Copiar Links de Afiliados
- Clique no ícone **📋 (copiar)** ao lado de cada link
- O link será copiado para sua área de transferência
- Use em suas redes sociais, blog ou WhatsApp

#### 2. Abrir Links Diretamente
- Clique no ícone **🔗 (link externo)** para abrir o produto
- Verifique se o link está funcionando corretamente
- Teste a experiência do usuário

#### 3. Buscar no TikTok
- Clique no botão **"Buscar"** para abrir o TikTok
- Use as palavras-chave sugeridas para encontrar conteúdo relacionado
- Inspire-se em vídeos existentes

#### 4. WhatsApp Direto
- Clique no botão **"WhatsApp"** para contato direto
- Mensagem pré-formatada com informações do produto
- Inclui todos os links de afiliados

## 🎵 Aba "Sugestões TikTok"

### O que você encontra:
- **Ideias de conteúdo** para cada produto
- **Hashtags otimizadas** para TikTok
- **Links de afiliados** para incluir na descrição
- **Estratégias de conteúdo** por categoria

### Como usar:

#### 1. Ideias de Conteúdo
- Escolha uma ideia que combine com seu estilo
- Adapte para sua audiência
- Mantenha a autenticidade

#### 2. Hashtags
- Clique nas hashtags para copiar
- Combine várias hashtags relevantes
- Use as sugeridas + suas próprias

#### 3. Links na Descrição
- Copie os links de afiliados
- Inclua na descrição do vídeo
- Use call-to-action atrativo
